# Guía para crear un Token Personal de GitHub

## Pasos para crear el token:

1. Ve a GitHub y haz clic en tu foto de perfil (esquina superior derecha)
2. Selecciona **Settings** (Configuración)
3. En el menú lateral izquierdo, baja hasta **Developer settings**
4. Haz clic en **Personal access tokens** → **Tokens (classic)**
5. Haz clic en **Generate new token** → **Generate new token (classic)**
6. Dale un nombre al token (ej: "HYDRAWARE2.0 - Desarrollo")
7. Selecciona el alcance (scope): marca **repo** (esto dará acceso a repositorios)
8. Haz clic en **Generate token** al final de la página
9. **¡IMPORTANTE!** Copia el token inmediatamente porque solo se muestra una vez

## Nota de seguridad:
- Guarda el token en un lugar seguro
- No compartas el token públicamente
- Si lo pierdes, tendrás que generar uno nuevo
