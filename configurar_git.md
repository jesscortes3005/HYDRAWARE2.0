# Configuración de Git para HYDRAWARE2.0

## Opción 1: Si tienes permisos en el repositorio original

Si el dueño del repositorio (jesscortes3005) te ha dado permisos de escritura:

1. Crea un token personal de GitHub (ver GUIA_TOKEN_GITHUB.md)
2. Cuando hagas `git push`, Git te pedirá usuario y contraseña:
   - Usuario: tu usuario de GitHub (jesscortes3005 o tu usuario)
   - Contraseña: usa el token personal (NO tu contraseña de GitHub)

## Opción 2: Trabajar con tu propio fork

Si no tienes permisos en el repositorio original, puedes:

1. Hacer un fork del repositorio en GitHub
2. Cambiar la URL remota a tu fork
3. Trabajar en tu fork y luego hacer un Pull Request al original

## Opción 3: Actualizar credenciales guardadas

Si ya tienes credenciales guardadas incorrectamente, puedes limpiarlas y volver a autenticarte.
